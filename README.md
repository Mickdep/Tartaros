## Installation

### Install Rust (if needed)
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
### Install Tartaros
```bash
git clone https://github.com/Mickdep/Tartaros.git
cd Tartaros
cargo build --release
```


## Usage

```
tartaros <ip_address>|<hostname>
```
